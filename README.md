# Image Segmentation in the presence of Intensity Inhomogeneity

## Overview
This is a python implementation of a level set method for image segmentation in the presence of intensity inhomogeneities.

## Referance
[C. Li, R. Huang, Z. Ding, J. C. Gatenby, D. N. Metaxas and J. C. Gore, "A Level Set Method for Image Segmentation in the Presence of Intensity Inhomogeneities With Application to MRI," in IEEE Transactions on Image Processing, vol. 20, no. 7, pp. 2007-2016, July 2011.](<https://10.1109/TIP.2011.2146190>)
